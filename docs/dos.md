# DOS command

## 常见内部命令
- cd // change directory 
- dir // show directory content
- md // create directory
- rd // remove directory
- copy A B  // copy dir/file A to dir/file B
- del // delete dir or file
- cls // clear screen
- ren // re-name dir/file
- COPY // copy dir/file
- c:    d:    e:   // 改变当前盘符
